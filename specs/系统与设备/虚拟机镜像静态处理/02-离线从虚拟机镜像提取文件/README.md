# 离线从虚拟机镜像提取文件

解决“我已经拿到磁盘镜像了，怎么把里面的指定文件或目录导出来”。

## 文件说明

- [`spec.yaml`](./spec.yaml)
  定义这个能力的输入、输出和流程骨架
- [`离线从虚拟机镜像提取文件-SPEC.md`](./离线从虚拟机镜像提取文件-SPEC.md)
  展开说明如何只读挂载、定位路径并导出文件

## 这个能力解决什么问题

这一步常见目标包括：

- 提取配置文件
- 提取日志
- 导出脚本、证书、密钥文件之外的普通业务文件
- 看系统目录结构是否完整

## 最小输出

- 使用的镜像文件
- 导出的文件或目录
- 导出路径
