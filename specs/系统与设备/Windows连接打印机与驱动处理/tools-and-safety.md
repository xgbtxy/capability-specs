# Tools And Safety

## 推荐工具路径

- 先用 Windows 自带“打印机和扫描仪”尝试添加
- 如果是网络打印机，优先按 IP 或主机名添加
- 如果驱动缺失，优先去厂家官网按准确型号下载
- 如果已经知道端口和队列要求，可用 PowerShell 做端口和打印机队列管理

## 安全与边界

- 不要优先使用第三方驱动下载站
- 不要在没确认型号前乱装相近型号驱动
- 不要把固件升级和驱动安装混在一起处理
- 如果是共享打印机，还要先确认共享主机或打印服务器是否正常

## 优先判断顺序

1. 先确认打印机是不是在线
2. 再确认地址和型号
3. 再确认驱动来源
4. 最后才是 Windows 里添加和验证
