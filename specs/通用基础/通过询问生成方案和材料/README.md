# 通过询问生成方案和材料

很多材料写不出来，不是不会写，而是信息没问清楚。

这组能力先通过一轮提问把关键信息收齐，再把内容整理成方案、汇报稿、研究材料或初稿。

## 推荐工具

- 大模型对话
- Markdown
- PPT / Word / 文档编辑器

## 最小使用方式

1. 先问清楚这份材料给谁看
2. 再问清楚要解决什么问题
3. 再问清楚现状、目标、约束、亮点和风险
4. 最后再让 AI 生成初稿或正式材料
