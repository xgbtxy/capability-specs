# 能力名称

一句话说清楚这个能力能解决什么问题。

写这个模板时，尽量记住这几件事：

- 目录名优先写用户能看懂的场景
- 底层尽量依赖强开源工具
- 内容尽量短、直接，像一个小能力包
- 不要试图把所有细节都写完，给大模型一个清晰结构就够了
