# Linux离线恢复登录访问

解决“我忘了 Linux 虚机的登录方式，但手里有离线镜像，怎么恢复进入系统”。

## 文件说明

- [`spec.yaml`](./spec.yaml)
  定义这个能力的输入、输出和流程骨架
- [`Linux离线恢复登录访问-SPEC.md`](./Linux离线恢复登录访问-SPEC.md)
  展开说明如何在离线状态下恢复访问而不是提取原密码

## 这个能力解决什么问题

这一步的目标不是“知道旧密码是什么”，而是：

- 手里有 Linux 镜像
- 需要重新获得登录入口
- 之后再进入系统设置新密码或修复认证方式

## 最小输出

- 目标镜像
- 采用的恢复方式
- 修改过的关键文件
- 是否完成最小验证
