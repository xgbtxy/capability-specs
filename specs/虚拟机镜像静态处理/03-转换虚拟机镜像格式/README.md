# 转换虚拟机镜像格式

解决“我手里这个镜像格式不方便后续处理，怎么把它稳定转换成另一种格式”。

## 文件说明

- [`spec.yaml`](./spec.yaml)
  定义这个能力的输入、输出和流程骨架
- [`转换虚拟机镜像格式-SPEC.md`](./转换虚拟机镜像格式-SPEC.md)
  展开说明如何识别源格式、选择目标格式和验证转换结果

## 这个能力解决什么问题

典型需求包括：

- `vmdk -> qcow2`
- `vhdx -> qcow2`
- `qcow2 -> raw`
- 从 `ova/ovf` 中拿到真实磁盘后再转换

## 最小输出

- 源镜像格式
- 目标镜像格式
- 转换后的文件路径
