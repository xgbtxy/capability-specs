## 这个能力解决什么问题

静态镜像处理最常见的起点问题不是工具不会用，而是：

- 不知道手里的文件是不是磁盘
- 不知道 `ovf` 和 `vmdk` 的关系
- 不知道单盘、多盘、目录型虚拟机该怎么判断
- 不知道下一步应该直接挂载、先提文件，还是先转换

## 最小输出

- 输入对象类型
- 真实可处理磁盘文件
- 推荐下一步动作
