# 确认打印机地址与连接方式

先搞清楚这台打印机到底怎么被找到，这是后面一切操作的起点。

## 解决什么问题

- 不知道打印机 IP 或主机名
- 不确定是 USB、局域网、Wi-Fi 还是共享打印机
- 还没开始装驱动，就已经不知道该连哪台机器

## 推荐工具

- Windows 网络信息与 `ping`
- 局域网扫描工具或路由器设备列表
- 打印机机身面板、网络配置页、贴纸标签

## 最小使用方式

1. 先确认打印机和电脑是不是在同一局域网
2. 去打印机机身或面板上看 IP、主机名、MAC 或型号
3. 如果拿不到，去路由器设备列表或扫描局域网
4. 记录最终要使用的 IP 或主机名
