# Playwright浏览器自动化

把“确认站点前提、编写自动化步骤、抓取导出和截图、验证结果与回退”拆成可复用的小能力。

## 推荐工具

- Playwright
  官方链接：[Playwright](https://playwright.dev/)

## 最小使用方式

1. 先确认站点、账号和前置条件
2. 用 Playwright 编写或录制步骤
3. 完成抓取、表单、截图、导出等动作
4. 检查输出结果并准备回退路径
