# jq处理JSON数据

把“看 JSON、筛字段、重组结构、批量处理 API 和日志结果”拆成可复用的小能力。

## 推荐工具

- jq
  官方链接：[jq](https://jqlang.org/) / [Manual](https://jqlang.org/manual/dev/)

## 最小使用方式

1. 先看清 JSON 长什么样
2. 用 jq 做筛选、提取、重组
3. 批量处理时先拿一条样本跑通
4. 保存输出并验证结果格式
