# environment-setup

这组能力至少要准备这些环境：

- 已经确认服务端真实连接参数
- 本机有对应客户端
- 如果是 Clash Verge，需要知道它的内核是 `Mihomo`

推荐工具：

- Xray
  官方链接：[Xray-core](https://github.com/XTLS/Xray-core)
- Clash Verge Rev
  官方链接：[clash-verge-rev](https://github.com/clash-verge-rev/clash-verge-rev)
- Mihomo
  官方链接：[mihomo](https://github.com/MetaCubeX/mihomo)

最小检查方式：

- 确认地址、端口、UUID、公钥、SNI、short-id 都正确
- 确认客户端支持对应协议和参数
