# 压缩包整理、加密与交付

把“识别压缩包、解压查看、压缩分卷加密、交付前验证”拆成可复用的小能力。

## 推荐工具

- 7-Zip
  官方链接：[7-Zip](https://www.7-zip.org/) / [Download](https://www.7-zip.org/download.html)

## 最小使用方式

1. 先确认压缩包类型和这次要做什么
2. 解压或查看内容
3. 需要时再压缩、分卷、加密
4. 交付前验证压缩包能否正常打开和还原
