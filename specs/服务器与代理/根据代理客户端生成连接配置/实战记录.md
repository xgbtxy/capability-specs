# 实战记录

已验证过一次真实场景：

- 给同一个 `Xray Reality` 节点生成了 `v2ray/Xray` 客户端 JSON
- 生成了 `Clash Verge` 的基础分流配置
- 手机端可以正常使用
- Windows 上 `Clash Verge` 开 `TUN` 时超时，最后定位到是客户端侧 `TUN` 路径问题

这次最值钱的经验：

- 同一节点在不同客户端上的可用性真的会不一样
- `Clash Verge + Mihomo + Windows TUN` 需要单独排
- 最后更稳的方向是：
  - 管理员权限运行
  - `stack: system`
  - `mtu: 1500`
  - `strict-route: true`
